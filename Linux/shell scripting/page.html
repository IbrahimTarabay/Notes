Shell Scripting

What is a Shell Script?
 A shell script is an executable file containing multiple shell commands that are executed sequentially. The file can contain:

1-Shell (#!/bin/bash)-->shell that will run in
2-Comments(# comments)
3-Commands(echo,cp,grep etc)
4-Statements(if,while,for etc)

Shell script should have executable permissions(-rwx r-x r-x)

Shell script has to be called from absolute path(ex: /home/userdir/script.bash)

If called from current loaction then run: ./script.bash
