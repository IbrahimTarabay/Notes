Create a central entry point to the framework





Using a front controller:

1-The URL does not map to an individual PHP script

2-All requests are sent through one page

3-This is called a Front Controller:
 -->Provides a central entry point for all requests
 -->Handles everything common to every request, Such as session handling etc

The request is in the query string

1-The query string is the part of the URL that comes after the first question mark
 -->daveh.io/index.php?/home

2-We can use this to decide where to route the request(which controller)
 -->daveh.io/index.php?/show_post/123
 -->daveh.io/index.php?/posts?page=2

3-The entire query string will be the request URL or route

Remove the default page

-->The default page is (most likely) index.php
-->So it can be removed from the URL:

daveh.io/index.php  = daveh.io
