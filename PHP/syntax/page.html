Increment & Decrement

$x++; // post-increment
$x--; // post-decrement
++$x; // pre-increment
--$x; // pre-increment

The difference is that the post-increment returns the original value before it changes the variable, while the pre-increment changes the variable first and then returns the value.

ex:
$a = 2; $b = $a++; // $a=3, $b=2
$a = 2; $b = ++$a; // $a=3, $b=3

Associative Arrays

Associative arrays are arrays that use named keys that you assign to them.There are two ways to create an associative array:

$people = array("David"=>"27","Amy"=>"21","John"=>"42");
//or
$people['David'] = "27";
$people['Amy'] = "21";
$people['John'] = "42";

Multi-Dimensional Arrays

-->A mulit-dimensional array contains one or more arrays

The dimension of an array indicates the number of indices you woud need to select an element:
1-for a two-dimensional array, you need two indices to select an element
2-for a three-dimensional array, you need three indices to select an element
