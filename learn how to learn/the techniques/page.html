System vs Goals

-->we need to have a clear defined SMART goals, goals are good but systems are even better

Success = Luck x Skills that you obtain

-->if you're an efficient learner you're not guaranteed success luck still plays a role in our lives but we have control over the skills that we obtain

-->so by having a system instead of a goal we actually can incorporate this luck side of the equation

examples:
1- i need to study 6 hours a day this is system

2-i need to finish this project after month this is a goal

-->by creating a system instead of a goal we think about the small chunks of time that we need so that we do things each day that have long term effects

-->if you find yourself being the type of person that doesn't accomplish their goals of finds procrastination easy then systems and processes is what you need

-->Systems eventually become into habits

-->Goals are really good for short and simple small tasks things that you can usually do in a day but if you want to accomplish something over your life instead of a goal have a system.
