masscan-->is built to scan the entire internet really fast so it's really fast port scanner, not recommended to use it for scan entire internet because you will have people knocking at your door pretty quick 

#masscan -p1-65535 --rate 1000 192.168.231.129 (--rate 1000 mean 1000 threads)


scan with metasploit:

#msfconsole

msf5> search portscan

msf5> use 4
msf5> use auxiliary/scanner/portscan/syn

msf5 auxiliary(scanner/portscan/syn) > options

msf5 auxiliary(scanner/portscan/syn) > set RHOSTS 192.168.231.129

msf5 auxiliary(scanner/portscan/syn) > set ports 1-65535

msf5 auxiliary(scanner/portscan/syn) > set threads 1000

msf5 auxiliary(scanner/portscan/syn) > run

-->say you get onto a machine and this machine doesn't have any nmap build into it but we can run metasploit through a shell that we have on a machine so we can run a port scan so that makes life a lot easier instead of install nmap or other tools and run scanning
