Optimization Strategy

1-Be specific and go long-tail
2-Pause keywords with low CTR
3-Use Match Types
4-Add Negative Keywords
5-Bundle closely related keywords in one Ad Group

Testing and Running Experiments

-->To figure out what contributes to the success of an ad, you can run so-called "single variable A/B split tests" in other words, testing two ads that only differ in one variable(such as the headline, description or landing page) against each other. By running a single variable A/B split test, you can learn how a single variable influences the performance of the campaign and which iteration(A or B) is better

To set up a test, follow these steps:

1-Define one goal metric
2-Identify the element(s) of the ad you want to test
-->elements that you can change:
-Headline
-Description
-Call to Action
-The promoted offer
-Landing Page
-URL Path
-Ad Extensions
-Ad Group Keywords

3-Create the different version(s) of your ads/ad groups
4-Changing Ad Copy
5-Analyze your results
