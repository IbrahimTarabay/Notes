What is a database?

-->it's a collection of data, a method for accessing and manipulating that data

Drum memory was a magnetic data storage device invented by Gustav Tauschek in 1932 in Austria. Drums were widely used in the 1950s and into the 1960s as computer memory. For many early computers, drum memory formed the main working memory of the computer.
 

How companies use a database??

1-it might have product managers and product managers always have to know the product that they are working on, they need to get data and learn about the products health, whether a product is working properly

2-Marketers, you need to find out information from data you want to analyze business decision, give you insights about how to market a product

3-Web developer, mobile app developer, that creates apps on phone or web and those apps usually have things like user signings or profiles that you need to store somewhere

4-Data analysts, data scientists that understand data, that analyze data and make decisions such as perhaps even building machine learning models

5-data engineers or database administrators, these are the people that actually help set up the databases and a company update software, install things, make sure that databases are connected with one another, they might use things like Hadoop, facebook's presto, google's big query, amazon redshift
